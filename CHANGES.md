## [1.0.0] - 2023-05-17

### Added

- Initial provider implementation
- PKCS\#11 key URI support (RFC 7512)
- ECDSA signature support (PKCS\#1, PSS)
- RSA sign support (PKCS\#1)
- RSA decrypt support (raw, PKCS\#1, OAEP)
- forwarding for all public key operations
